제작 동기

: 2020년도의 경우 코로나19로 인해서 오프라인 수업을 듣지 않는 신입생들이 많이 존재했고, 수업 이외에도 다양한 시설을 이용하지 못했다는 점으로 인해 교내에 어떤 건물과 장소가 있는지 모르는 경우를 보고는 했습니다. 또한, 학교에 막 입학한 새내기의 경우 본교에 있는 건물과 장소에 대해서 알 수 있는 하나의 플랫폼이 딱 정리되어 있지 않아 한눈에 바라보기 어렵다는 점도 있었습니다. 그래서 이번 '앱프로그래밍' 과목의 팀프로젝트에서 '한동대학교 캠퍼스 소개 사이트'를 만들어 서비스를 제공하고자 본 사이트를 제작하게 되었습니다.

[ 과목 설명 ]
앱 프로그래밍은 교양선택필수과목으로 꼭 들어야하는 과목 중 하나입니다. 앱 프로그래밍에서 우리는 Html, CSS, Javascript에 대해서 학습합니다. 또한, 팀프로젝트로 홈페이지를 만들어보면서 많은 프로그래밍 경험을 쌓을수도 있습니다. 권장 선수과목은 ICT응용입문 또는 소프트웨어입문이고 위 사진에 나와있는 김군오 교수님이 지도하는 강의입니다.

[ 추가 설명 ]
우리는 이 강의를 통해 동적인 웹페이지를 구축할 수 있게 됩니다. 학기가 시작되는 1주차부터 3주차까지는 웹 페이지 개발에 필요한 기초언어인 HTML에 대해서 학습합니다. 이 때에는 HTML이 정확하게 어떤 것인지에 대한 개념정의와 코드의 기초들에 대해서 배웁니다. 그리고 4주차부터 5주차까지는 웹 페이지에 스타일을 적용하는 CSS에 대해서 학습합니다. CSS 강의를 통해 글씨를 크게 하거나 색을 넣는 등 웹페이지를 디자인하는 방법을 학습합니다. 그리고 6주차부터 15주차까지는 동적 웹페이지 작성을 위해 javascript를 학습하는데 변수부터 연산자와 형변환, 더 나아가 조건문, 반복문, 함수에 대해 학습하고 core library라는 심화과정까지 학습하게 됩니다.
